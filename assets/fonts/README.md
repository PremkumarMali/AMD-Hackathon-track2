# Brand fonts

Drop the **Mortane** font file(s) here. Any of these work:

```
Mortane.otf
Mortane.ttf
Mortane.woff2        ← best for web (smallest)
Mortane-Regular.woff2
Mortane-Bold.woff2
Mortane-Italic.woff2
```

The app auto-detects any file whose name contains "Mortane" and embeds it
(base64) into the page — no config, works offline. If no file is present,
the UI falls back to an elegant serif stack (Playfair Display → Georgia →
serif) so nothing breaks.

> Mortane is a commercial font — use your licensed copy. To convert an
> `.otf`/`.ttf` to `.woff2`, use any "font to woff2" converter.
